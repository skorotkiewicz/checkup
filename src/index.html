API Endpoints

Endpoint Description
/repo/{host}/{owner}/{repo} Get releases (HTML) for GitHub/GitLab
/forgejo/{host}/{owner}/{repo} Get releases (HTML) from any Forgejo instance
/repo/{host}/{owner}/{repo}/cache Get cached releases (JSON)
/repo/{host}/{owner}/{repo}/latest.{ext} Redirect to latest release asset

Latest Asset Downloads

Get the latest version of any asset with a consistent URL:

/repo/github.com/owner/repo/latest.tar.gz
/repo/github.com/owner/repo/latest.AppImage
/repo/github.com/owner/repo/latest.exe

Perfect for scripts and CI/CD pipelines - the URL stays the same even when new versions are released.

Examples

GitHub

/repo/github.com/sharkdp/bat

GitLab

/repo/gitlab.com/gitlab-org/gitlab

Codeberg (Forgejo)

/forgejo/codeberg.org/forgejo/forgejo
